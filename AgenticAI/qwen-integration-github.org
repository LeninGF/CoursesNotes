* ¿Qué es Qwen?
Qwen es un modelo de lenguaje de gran tamaño (LLM) desarrollado por
Alibaba Cloud. Es capaz de comprender y generar texto en múltiples
idiomas, incluyendo español. Ofrece capacidades avanzadas de
razonamiento, programación y análisis.

* ¿Funcionalidades de Qwen?
- Generación de código en diversos lenguajes de programación.
- Respuesta a preguntas complejas y resolución de problemas.
- Traducción y comprensión de textos multilingües.
- Asistencia en tareas de depuración y optimización de código.
- Integración con herramientas de desarrollo como GitHub.

* Qwen Coder
Qwen Coder es una versión especializada de Qwen enfocada en tareas de
programación. Permite generar, explicar y depurar código
automáticamente. Ideal para desarrolladores que buscan aumentar su
productividad mediante asistencia inteligente en la escritura y
mantenimiento de software.
* Integración con Github
1. Login en [[https://coder.qwen.ai/]]
2. Abrir Codex
3. Sincronizar con Github y Autorizar
   1. Un repositorio específico
   2. Todos los repositorios
4. Añadir Entorno (/Environment/): Despliega la organización y los
   repositorios a los que se puede acceder. Seleccionar el repositorio
   y dar en /Submit/. 
5. Retornar a la vista inicial de [[https://coder.qwen.ai/]]
6. Seleccionar el entorno y colocar un prompt
